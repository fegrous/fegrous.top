---

## ✨ 特性

- 🌓 **自适应亮暗模式** — 跟随系统主题自动切换
- ⏳ **时间胶囊** — 实时显示今日、本周、本月、今年进度
- 📱 **响应式设计** — PC / 手机 / 平板都适配
- ⚡ **纯静态，超快加载** — 零依赖，一个 HTML 搞定
- 🔗 **社交链接** — GitHub、Telegram、Bilibili 一键直达
- 🖼️ **项目展示** — 精选项目卡片

## 🛠️ 技术栈

- **HTML5** — 语义化结构
- **CSS3** — 自定义属性 + 渐变 + Flexbox
- **Vanilla JS** — 纯原生，无框架

## 🚀 本地预览

```bash
# 克隆
git clone git@github.com:fegrous/fegrous.top.git
cd fegrous.top

# 任意 HTTP 服务器预览
python3 -m http.server 3000
# 或
npx serve .
```

打开浏览器访问 `http://localhost:3000` 即可。
